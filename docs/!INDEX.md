# Documentation

This is the top-level documentation index and rulebook for Space Rocks docs.

## Rulebook

- Normal documentation folders use `!INDEX.md` as the folder index.
- `stubs/` folders are exempt from the `!INDEX.md` index requirement.
- Top-level docs should stay navigational and policy-oriented, not duplicate full policy detail.
- Legacy docs are migration-only and should not be treated as current authority.
