# Server

Parent index: [Devtools](../!README.md)

Server devtools documentation lives here.

## Ownership

This folder owns docs for server-side development and debug tooling.

## Does Not Belong

- Production gameplay docs.
- Domain flow docs.
- Planning docs.
- Parallel debug-only gameplay logic that bypasses real game seams.
- Stub content as canonical devtools authority.
