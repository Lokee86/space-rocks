# Web

This folder owns web service implementation planning for Space Rocks.

It covers the future `services/web/` implementation, including the V0 devlog static site and any service-level web runtime decisions that belong under service planning.

Parent index: [Planning](../!INDEX.md)

## Ownership

This folder is owned by web-service planning.
Use it for implementation shape, route structure, static generation, content loading, layout integration, and other service-specific web concerns.

## Does Not Belong

- Web-domain product direction.
- Deployment or hosting mechanics.
- API contracts.
- Generated runtime artifacts.
- Stub content once it graduates into a direct service-planning file.
