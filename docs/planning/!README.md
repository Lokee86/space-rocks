# Planning

`docs/planning` contains planning docs for future, unresolved, proposed, or not-yet-current work.

## Ownership

This folder owns the planning index for work that has not yet become current documentation.

## Does Not Belong

- Current canonical docs.
- Legacy docs as current authority.
- Stub content as canonical planning authority.
