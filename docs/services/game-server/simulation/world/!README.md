# Game Server Simulation World

Parent index: [Game Server Simulation](../!README.md)

World simulation documentation for the game server lives here.

## Ownership

This folder owns toroidal space, motion, visibility and despawn, asteroid spawning and variants, destruction, and collision-shape and physics support documentation for the game server simulation.

## Does Not Belong

- Player session ownership.
- Room membership or lifecycle rules.
- WebSocket transport details.
- External integration internals.
- Process startup or shutdown.
