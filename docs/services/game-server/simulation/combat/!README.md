# Game Server Simulation Combat

Combat documentation for the game server lives here.

## Ownership

This folder owns weapons and projectile fire, collision-to-damage flow, damage resolution, scoring, counter mutation, and radial effects documentation for the game server simulation.

## Does Not Belong

- Room membership or lifecycle rules.
- WebSocket transport details.
- External integration internals.
- Process startup or shutdown.
